# My Omarchy supplements
## description
  This is my repo to maintain a backup of my working omarchy setup
## Purpose
  The purpose of this repo is to get a quick way to restore your omarchy setup in case of a broken update of just like a quick way of startup a working and according to your preferences environment for a new machine
